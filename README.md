# ИЛИ Studio

Генератор вирусных вертикальных видео «Что выберешь?» (красное или синее) для TikTok / Reels / Shorts. Полный пайплайн: сценарий → картинки → озвучка → композиция → MP4.

- **DeepSeek (deepseek-v4-pro)** пишет русский сценарий по референс-стилю (еда, абсурдные дилеммы, интерактив «Поделиться в Telegram», CTA-вопросы) — всё редактируется в UI.
- **Pexels / Openverse** — автоподбор картинок (с кандидатами и заменой).
- **ElevenLabs** (голос Adam, `eleven_multilingual_v2`) — озвучка с таймстемпами: вторая картинка появляется ровно на слове «или».
- **HeyGen Hyperframes** — HTML-композиция 1080×1920 с GSAP-анимациями, тикающими часами, процентами, вжухами и попами; рендер в MP4 через облачный API или локально.

## Запуск

```bash
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000), вставь API-ключи (DeepSeek, ElevenLabs, HeyGen, опц. Pexels) в форму (ключи уходят только в серверный роут конкретного запроса; галочка «запомнить» хранит их в localStorage браузера, в репозиторий ничего не пишется).

## Пайплайн

1. **Сценарий** — тема + количество вопросов → «Сгенерировать сценарий». Каждый вопрос: текст озвучки, подписи на экране, поисковые запросы картинок, проценты, время на раздумья — всё редактируемое.
2. **Картинки** — «Подобрать все картинки» (Pexels с ключом или Openverse без ключа). Клик по миниатюре — выбор из кандидатов или новый поиск.
3. **Озвучка** — «Озвучить всё (Adam)». Тайминг слова «или» берётся из character-timestamps ElevenLabs.
4. **Сборка** — генерируется самодостаточный Hyperframes-бандл (`index.html` + ассеты + звуки + GSAP) и интерактивное превью прямо в браузере (play/pause/перемотка).
5. **Рендер** — кнопкой через HeyGen Hyperframes API (9:16, 1080p, mp4), либо скачай ZIP и рендерь локально:

```bash
unzip <project>-hyperframes.zip -d my-video && cd my-video
npx hyperframes render   # нужны Node 22+, Chrome, ffmpeg
```

## Формат видео

Скопирован с вирусных референсов покадрово:

- красная/синяя половины, чёрный разделитель и бейдж «ИЛИ»;
- вариант А влетает справа под «поп», вариант Б появляется на слове «или»;
- после вопроса — тикающие часы (~3.5 с) на раздумья;
- опциональный показ процентов (зелёный победитель / красный проигравший) с «дзынь»;
- вжух-переход между вопросами;
- водяной знак с твоим ником (переопределяется переменной `handle` при рендере).

Звуки (тиканье, вжух, поп, дзынь) синтезированы и лежат в `public/sfx` — роялти-фри.
